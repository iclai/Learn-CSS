# CSS 3.0基礎

CSS是「Cascading Style Sheets」的縮寫，中文意思是「**串接樣式表、層疊樣式表**」，CSS就像前面提過的幫網頁穿上漂亮的衣服，凡是網頁看的到的圖片、文字、表格......都可以套用。

CSS3.0是CSS2.0的進化版，也就是 CSS2.0和CSS3.0是沒有差別的，CSS3只是CSS2.0一個擴展的語法，所以還是要有基礎的CSS觀念，才能編寫CSS3.0
