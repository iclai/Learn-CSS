# box model

### Padding / margin

內距與外距是CSS中很常用到的屬性。

藍色框代表網頁內容，margin是代表以內容物為中心，往外擴張，padding則是以內容物為中心，往內縮。

![](.gitbook/assets/image%20%2818%29.png)

margin 和 padding各別設定:

```css
/** margin **/
margin-top: 值;
margin-right: 值;
margin-bottom: 值;
margin-left: 值;

/** padding **/
padding-top: 值;
padding-right: 值;
padding-bottom: 值;
padding-left: 值;

```
